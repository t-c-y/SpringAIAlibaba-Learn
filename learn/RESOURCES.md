# Spring AI Alibaba Resources

## Knowledge

- [Spring AI Alibaba 官方文档](https://java2ai.com/docs/1.0.0.2/overview/)
  Spring AI Alibaba 的主要学习入口。Use for: 框架定位、Starter、DashScope、ChatBot、RAG、Graph、MCP 等课程主题。
- [Spring AI Alibaba GitHub 仓库](https://github.com/alibaba/spring-ai-alibaba)
  官方源码仓库。Use for: 核对模块名称、版本、自动配置和真实实现。
- [Spring AI Alibaba Examples](https://github.com/spring-ai-alibaba/examples)
  官方示例仓库。Use for: Demo 写法、依赖组合、配置方式和功能边界。
- [Spring AI 官方参考文档](https://docs.spring.io/spring-ai/reference/)
  Spring AI 的基础抽象文档。Use for: ChatClient、Prompt、ChatModel、Tools、Vector Store、RAG 等底层概念。
- [阿里云百炼 / Model Studio 文档](https://help.aliyun.com/zh/model-studio/)
  DashScope / 百炼服务侧文档。Use for: API Key、模型名称、开通权限、模型能力和服务侧限制。
- [通义千问模型文档](https://help.aliyun.com/zh/model-studio/user-guide/qwen)
  Qwen 模型能力说明。Use for: 选择模型、理解输入输出限制、规划企业场景模型策略。
