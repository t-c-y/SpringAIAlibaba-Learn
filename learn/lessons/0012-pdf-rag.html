PDF 知识库：按页引用与上传灌库

本课目标：会用 PagePdfDocumentReader 处理 PDF，让回答能标注 [page N] 引用。

配套 Demo：demos/13-pdf-rag。参考卡：PDF RAG 快速参考。

关键差异

  • 相比 TextReader，PagePdfDocumentReader 会在 Document metadata 上写入 page_number。
  • System Prompt 里可要求模型在回答末尾用 [page N] 引用来源，实现可追溯。
  • 扫描版 PDF 需要先 OCR，本课不覆盖。

核心代码

PagePdfDocumentReader reader = new PagePdfDocumentReader(pdf,
    PdfDocumentReaderConfig.builder()
        .withPagesPerDocument(1)
        .build());
List<Document> docs = reader.get();
docs.forEach(d -> d.getMetadata().put("file_name", fileName));
vectorStore.add(new TokenTextSplitter(400, 200, ...).apply(docs));

动手

  1. 上传一份产品手册 PDF：curl -F 'file=@x.pdf' /pdf/upload
  2. 问一个只有 PDF 里才有的问题，观察回答有 [page N]。
  3. 调 /pdf/retrieve，看命中的 page 是否与回答一致。
