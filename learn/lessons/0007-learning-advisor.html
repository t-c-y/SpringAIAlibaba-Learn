学习顾问 Memory：短期会话 + 长期画像

本课目标：在多轮记忆基础上叠加"用户画像"这一层长期记忆，理解两级记忆的分工。

配套 Demo：demos/08-learning-advisor。参考卡：学习顾问快速参考。

两层记忆

短期 / 会话

  • ChatMemory 存储
  • conversationId 隔离
  • "这次会话说过的话"

长期 / 画像

  • LearnerProfileStore 存储
  • userId 隔离
  • "稳定的用户属性"

怎么把画像给模型

不是每轮都塞 user 消息，而是**渲染进 System Prompt**：

String system = """
你是学习顾问，服务对象画像：

%s

回答规则：...
""".formatted(profile.toSystemFragment());

动手

  1. 建画像：POST /advisor/profile
  2. 问"下周该学什么？"，观察建议是否基于画像。
  3. 不建画像直接问，对比差异。
  4. 修改画像的 targetRole，再问同一问题。
