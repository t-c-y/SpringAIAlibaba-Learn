Tool Calling 入门：计算器

本课目标：第一次让模型"调用你写的 Java 方法"，理解 @Tool 描述如何影响模型选择。

配套 Demo：demos/09-calculator-tool。参考卡：Tool Calling 快速参考。

信息流

用户 → ChatClient → LLM
                    │  选工具、传参数
                    ▼
                  你的 Java 方法（@Tool）
                    │  返回结果
                    ▼
                  LLM 综合成自然语言

核心注解

@Tool(description = "两个数相加，返回 a+b")
public double add(@ToolParam(description = "第一个加数") double a,
                  @ToolParam(description = "第二个加数") double b) { return a + b; }

// 注册
ChatClient c = builder.defaultTools(new CalculatorTools()).build();

description 是模型选工具的主要依据，务必写清楚"什么场景下用"。

动手

  1. 启动 demos/09-calculator-tool
  2. 问 (18+27)*3/5，观察日志能看到多次 tool call。
  3. 把 description 改模糊，看模型是否还会用它。
  4. 把 divide 里除数为 0 抛异常改成返回错误 record，对比效果。

动手验证：curl 快速调用

前置条件：export DASHSCOPE_API_KEY=你的 API Key。启动对应 Demo（每个接口块第一行给出目录和端口），下面命令均可直接复制执行。

🔹 09-calculator-tool（端口 8088）

启动：cd demos/09-calculator-tool && mvn spring-boot:run

  1. 四则运算：模型自动判断何时调用 add/subtract/multiply/divide 工具

    curl -X POST http://localhost:8088/calc/ask \
  -H 'Content-Type: application/json'\
  -d '{"question":"(3.14 + 2.86) * 5 等于多少？再减去 4.5 呢"}'
# 预期：{"success":true,"answer":"...模型返回的中文回答..."}

⚡ 3 条命令快速验证 09-calculator-tool

curl -X POST http://localhost:8088/calc/ask -H 'Content-Type: application/json' -d '{"question":"(3.14 + 2.86) * 5 等于多少"}'
