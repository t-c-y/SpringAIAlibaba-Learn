Tool Calling 入门：计算器

本课目标：第一次让模型"调用你写的 Java 方法"，理解 @Tool 描述如何影响模型选择。

配套 Demo：demos/09-calculator-tool。参考卡：Tool Calling 快速参考。

信息流

用户 → ChatClient → LLM
                    │  选工具、传参数
                    ▼
                  你的 Java 方法（@Tool）
                    │  返回结果
                    ▼
                  LLM 综合成自然语言

核心注解

@Tool(description = "两个数相加，返回 a+b")
public double add(@ToolParam(description = "第一个加数") double a,
                  @ToolParam(description = "第二个加数") double b) { return a + b; }

// 注册
ChatClient c = builder.defaultTools(new CalculatorTools()).build();

description 是模型选工具的主要依据，务必写清楚"什么场景下用"。

动手

  1. 启动 demos/09-calculator-tool
  2. 问 (18+27)*3/5，观察日志能看到多次 tool call。
  3. 把 description 改模糊，看模型是否还会用它。
  4. 把 divide 里除数为 0 抛异常改成返回错误 record，对比效果。
