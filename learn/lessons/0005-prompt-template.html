Prompt 模板管理：把 Prompt 变成可维护资源

本课目标：会用 PromptTemplate / SystemPromptTemplate，把 Prompt 从字符串硬编码升级为 .st 资源文件。

配套 Demo：demos/06-prompt-template。参考卡：Prompt 模板快速参考。

为什么要模板化

  • Prompt 是"产品"，写 Prompt 的可能是产品经理、运营、法务，不都是工程师。
  • 要能 diff / review / 灰度 / 国际化。硬编码在 Java 里做不到。
  • 把 Prompt 放 src/main/resources/prompts/*.st，编辑不用重编代码，做 hot reload 也更容易。

两个组件

Message system = new SystemPromptTemplate(text)
    .createMessage(Map.of("role","学习助手","style","严谨"));

String user = new PromptTemplate(userText)
    .render(Map.of("topic","ChatMemory","level","入门"));

Prompt p = new Prompt(List.of(system, new UserMessage(user)));
chatClient.prompt(p).call().content();

模板用 {name} 占位符（StringTemplate 语法）。

目录结构

├── lesson-assistant-system.st
