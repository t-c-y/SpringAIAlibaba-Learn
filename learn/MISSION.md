# Mission: Spring AI Alibaba 企业级开发

## Why
你希望系统学习 Spring AI Alibaba，并把它用于企业级 AI 应用开发。学习重点不是 Java 或 Spring Boot 基础，而是把已有 Java 后端能力迁移到 ChatBot、Prompt、Memory、Tool Calling、RAG、Agent 编排与生产化治理上。

## Success looks like
- 能独立搭建 Spring AI Alibaba 应用，并通过 DashScope / 百炼调用大模型。
- 能开发企业常见 AI 能力：ChatBot、流式输出、结构化输出、工具调用、RAG 知识库问答。
- 能把 AI 能力封装成符合后端工程习惯的 Controller、Service、配置和测试。
- 能理解企业级 Agent 的关键边界：权限、安全、审计、观测、评测、成本和人工确认。
- 能在 `demos/` 目录中持续沉淀可运行、可复习、可扩展的课程 Demo。

## Constraints
- 使用中文讲解。
- 以代码实战为主，每个阶段尽量在 `demos/` 中形成一个可运行 Demo。
- 用户熟悉 Java / Spring Boot 技术栈，了解过 LLM 和 RAG，但 Spring AI Alibaba 从零开始。
- 课程可以采用问答式教学，优先保持每节课短小、可完成、有反馈。

## Out of scope
- 不重复讲解 Java 基础、Maven 基础、Spring Boot Web 基础。
- 不一开始追求复杂多 Agent 平台，先按 ChatClient → Prompt → Memory → Tool Calling → RAG → 生产化逐步推进。
