MCP 快速参考

两个核心方法

GET  /mcp/tools    → [{name, description, inputSchema}]
POST /mcp/call     ← {toolName, arguments} → 结构化结果

响应协议

模型每步只输出：
  {"tool":"list_files","args":{"dir":"/"}}
  或 {"final":"给用户的中文回答"}

生产要点

  • Server 端加鉴权 / 限流 / 审计。
  • Client 端限制循环最大步数。
  • 写操作走 dryRun preview + 人工确认。
