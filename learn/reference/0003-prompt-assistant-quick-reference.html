Prompt 学习助手快速参考

复习卡：三段式 System Prompt + 温度 + 业务字段。

三段式模板

# 角色
你是<某角色>，服务<某听众>。
# 输出结构
1) 一句话结论
2) 三点要点
3) 最小代码
4) 常见坑
# 边界
- 不确定要明说
- 只回答与 XX 有关的问题

建议参数

结构化 / 客服 / 政策

  • temperature: 0.1 ~ 0.3
  • 输出要"稳"

创意 / 头脑风暴

  • temperature: 0.7 ~ 1.0
  • 输出要"散"

快速自检

  • System Prompt 里有没有"角色 + 结构 + 边界"三段？
  • 业务变量是不是留在了 user 消息？
  • 温度是不是与目的匹配？
  • 是否给了明确的"不知道就说不知道"边界？

常见坑

  • 把"用户名字/日期"这类易变字段塞进 System Prompt → 会一直污染下去。
  • 没有"边界"段 → 模型容易越界胡编。
  • 结构太多层 → 模型抓不住重点，反而输出稀里糊涂。
