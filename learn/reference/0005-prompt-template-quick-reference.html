API 速查表

API / 概念一句话说明
new PromptTemplate("text {var}")内联字符串模板，{var} 是变量
new PromptTemplate(Resource)从 classpath 文件加载 .st 模板
.render(Map.of("k","v"))渲染为 String（单 message）
new SystemPromptTemplate(st).createMessage(vars)构造 System 消息
new UserPromptTemplate(st).createMessage(vars)构造 User 消息
classpath:prompts/xxx.st推荐模板资源路径

典型代码模板

var st = new PromptTemplate(new ClassPathResource("prompts/lesson-user.st"));
String userMsg = st.render(Map.of("topic", topic, "audience", audience));

踩坑清单

  • 不要用字符串拼接代替模板——易被 Prompt 注入（用户输入中的 { 和换行破坏 Prompt）
  • 模板文件保存为 UTF-8，避免 IDE 存成 GBK
  • 变量缺失会抛异常，render 前要校验必填字段
  • 渲染后总长度要算 token 避免超上下文
  • 不要在模板里硬编码 API Key/数据库密码
  • 多语言场景用不同 .st 文件（如 lesson-user.st / lesson-user-en.st）不要在模板里 if/else
