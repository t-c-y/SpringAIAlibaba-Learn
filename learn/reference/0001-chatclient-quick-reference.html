API 速查表

API / 概念一句话说明
chatClient.prompt()开始构造一次模型调用
.user(String)设置用户本次消息；Web 接口通常来自 request body
.system(String)设置本次 system message；defaultSystem 更常用
.defaultSystem(String)ChatClient 构建时设置，每次调用都生效
.call()同步调用；等待完整回答返回
.stream()流式调用，返回 Flux
.content()从 CallResponseSpec 取回答文本
.entity(Class)结构化输出为单个 Bean（见 0004）

典型代码模板

String answer = chatClient.prompt()
        .user("请介绍 Spring AI Alibaba")
        .call()
        .content();

踩坑清单

  • API Key 不要写在 application.yml，优先环境变量 DASHSCOPE_API_KEY
  • 模型名用 ${DASHSCOPE_CHAT_MODEL:qwen3.7-max} 占位符支持覆盖
  • defaultSystem 里的规则所有调用都会遵守，不要写死版本相关的业务数字
  • 调用 LLM 要设置超时（默认 30s 在弱网下可能不够）
  • ChatClient.Builder 是 Spring Boot Starter 自动注册的 Bean，不要自己 new ChatClient
