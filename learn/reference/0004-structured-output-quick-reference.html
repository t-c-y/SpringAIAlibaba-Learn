结构化输出快速参考

三种典型用法

// 1. 单对象
X x = client.prompt().user("...").call().entity(X.class);

// 2. 列表
List<X> list = client.prompt().user("...").call()
   .entity(new ParameterizedTypeReference<List<X>>() {});

// 3. Map
Map<String, Object> map = client.prompt().user("...").call()
   .entity(new ParameterizedTypeReference<Map<String, Object>>() {});

踩坑清单

  • 字段名不一致 → 模型输出的 JSON 字段跟 record 对不上 → 转换失败或字段为 null。
  • 温度太高 → 模型加解释或包 ```json``` → 反序列化异常。
  • 字段类型用 Object → 模型可能给字符串给对象 → 类型不稳定。
  • Java 17 record 是不可变的 → 反序列化只会走 canonical 构造器。

自检

  • 是否为 record？字段类型是否具体？
  • temperature ≤ 0.2？
  • System Prompt 里有没有加"不要输出 ```json``` 包装、不要解释"？
