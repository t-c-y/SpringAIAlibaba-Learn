API 速查表

API / 概念一句话说明
.entity(MyRecord.class)单个 Bean 结构化输出，底层用 BeanOutputConverter
.entity(new ParameterizedTypeReference>(){})集合/嵌套类型结构化输出
@JsonPropertyDescription("...")给字段加自然语言注释，会出现在 JSON Schema 里帮助模型理解
spring.ai.dashscope.chat.options.temperature建议 ≤ 0.1，高温度会让模型改字段名
BeanOutputConverter手动用：converter.getJsonSchema() 注入 System Prompt，再 converter.convert(output)

典型代码模板

.user(request.topic())
.call()
.entity(LessonOutline.class);
// record LessonOutline(String title, String summary, List sections, String difficulty) {}

  • 返回类型必须是具体 class 或具名 ParameterizedTypeReference，不能用 List
