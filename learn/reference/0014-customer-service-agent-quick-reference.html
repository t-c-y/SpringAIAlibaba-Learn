Memory+RAG+Tool 快速参考

组装

builder.defaultAdvisors(
    MessageChatMemoryAdvisor.builder(mem).build(),
    QuestionAnswerAdvisor.builder(vs)
        .searchRequest(SearchRequest.builder().topK(4).similarityThreshold(0.3).build())
        .build())
.defaultTools(tools)
.build();

分层建议

  • 知识类问题 → RAG。
  • 业务操作 → Tool + 服务端权限校验。
  • 会话上下文 → Memory。
  • 写操作 → 显式 pending 状态 + 人工审批。
