外部 API 风格 Tool 快速参考

返回结构模板

record CurrentWeather(
  boolean supported,     // 未支持城市 / 数据缺失时 false
  String cityCode,
  String cityName,
  Integer temperatureC,
  String condition,
  Integer humidityPercent,
  Integer windLevel,
  String error) {}

迁移到真实 API

  • 用 RestClient / OkHttp，注意 connect / read / write 超时都要显式设置。
  • 加缓存（Caffeine，例如同城 5 分钟内复用）。
  • 把 API Key 用配置注入到 Tool 类，不通过 Prompt 传。
  • 把错误码到用户话术的映射放 Tool 内部，不要暴露给模型。
