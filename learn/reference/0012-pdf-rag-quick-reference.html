PDF RAG 快速参考

依赖

<dependency>
  <groupId>org.springframework.ai</groupId>
  <artifactId>spring-ai-pdf-document-reader</artifactId>
</dependency>

两种 Reader

PagePdfDocumentReader

按页产生 Document，元数据带 page_number。

ParagraphPdfDocumentReader

按段落切分，保留章节结构（适合结构化手册）。

Prompt 引用模板

每条结论后用 [page N] 标注页码；
若涉及多页请合并成 [page 3,5]。
