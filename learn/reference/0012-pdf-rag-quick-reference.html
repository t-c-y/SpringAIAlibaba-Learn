API 速查表

API / 概念一句话说明
PagePdfDocumentReader一页一个 Document，metadata 带 page_number
ParagraphPdfDocumentReader按段落切（自动识别空行/缩进），更语义化
PdfDocumentReaderConfig配置页边距、按页还是按段落、是否提取字体信息
MultipartFileSpring MVC 上传 PDF 字段
PagePdfDocumentReader.read()返回 List，每个 Document 带 metadata.page_number

典型代码模板

@PostMapping("/pdf/upload")
public Object upload(@RequestParam("file") MultipartFile file) {
  var reader = new PagePdfDocumentReader(
      new ByteArrayResource(file.getBytes()), PdfDocumentReaderConfig.defaultConfig());
  var chunks = new TokenTextSplitter(400,200,10,5000,true).split(reader.read());
  vectorStore.add(chunks);
  return Map.of("success",true,"chunks",chunks.size());
}

踩坑清单

  • 扫描版 PDF（图片）PagePdfDocumentReader 抽不到文字，必须先 OCR
  • 加密/密码保护 PDF 要先解密
  • 跨页表格会被切断，overlap 只能部分缓解
  • 页码从 1 开始不是 0，System Prompt 要说明引用 [page N] 格式
  • MultipartFile 要做大小限制（≤20MB）和类型校验（只允许 application/pdf）
  • VectorStore 中要存文档 ID/版本，重复上传相同 PDF 要先删旧 chunks
  • OCR 场景推荐用 PaddleOCR/Azure Doc Intelligence/百炼文档理解
