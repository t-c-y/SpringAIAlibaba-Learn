业务 Tool + 权限快速参考

身份注入

// Controller
UserContext.set(userId);
try { ... } finally { UserContext.clear(); }

// Tool
String uid = UserContext.get();

权限层次

  1. 网关：认证（Auth）。
  2. Controller：授权（Authorization）。
  3. Tool：数据可见性（Data-level）。
  4. 业务：可审计的操作日志。

写操作模式

1. Tool 只“提交待办”，返回 status=pending。
2. 用户在业务系统里点确认，才真正执行。
3. 提交时带幂等 key，避免多次执行。
