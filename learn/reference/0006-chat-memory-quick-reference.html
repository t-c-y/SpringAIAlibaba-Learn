ChatMemory 快速参考

关键组件

ChatMemory

存储抽象；InMemoryChatMemory 是教学版，生产建议 Redis / DB。

Advisor

MessageChatMemoryAdvisor 负责读写记忆。

关键参数

  • CHAT_MEMORY_CONVERSATION_ID_KEY：会话标识，必须由服务端计算，不允许前端裸传。
  • CHAT_MEMORY_RETRIEVE_SIZE_KEY：每轮取回历史消息数上限，防上下文爆炸。

生产扩展方向

  • Redis-backed ChatMemory：多实例共享，加 TTL。
  • Token 上限：超过阈值触发摘要（summarize）压缩。
  • PII 脱敏：写入前对手机号、身份证等做 masking。
  • 可审计：记忆需要能被合规团队检索到。
