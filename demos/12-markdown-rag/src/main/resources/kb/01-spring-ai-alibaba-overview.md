# Spring AI Alibaba 概览

Spring AI Alibaba 是阿里巴巴基于 Spring AI 生态推出的一套企业级 AI 应用开发框架。它把 Spring Boot 的 starter 化配置、Bean 装配和 REST 支持带到大模型场景，让 Java 开发者可以像写传统业务一样开发 AI Agent。

核心亮点：

1. 无缝对接 DashScope / 百炼平台，同时兼容 OpenAI 协议。
2. 提供 ChatClient、Advisor、Tool Calling、RAG、Graph、MCP 等抽象。
3. 与 Spring Cloud / Nacos / Sentinel 结合，支持大规模生产部署。

当前学习的版本号是 1.0.0.2，基于 Spring Boot 3.3.5、Spring AI 1.0.x。
